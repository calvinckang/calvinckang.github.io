Cruise Connect

Website

Cruise Connect is a coach transfer service for cruise passengers. The current Cruise Connect website (www.intercruises.com/cruiseconnect) desperately needs to be redesigned — it's not responsive and not user-friendly. The main goal is to make the user can easily book a transfer. The users of this website are cruise passengers, mostly 50+ years old so the design needs to be adapted to them — simple, straightforward process and no tiny text. The new website is currently under development and will be launched soon.
